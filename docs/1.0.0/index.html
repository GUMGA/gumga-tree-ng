Descrição

O componente GumgaTreeNg foi criado com o objetivo de poder trabalhar com estrutura de dados de árvores.

Observações

Com este componente é possível manipular um array de objetos de forma interativa e simples. Utilizando o recurso drag-drop, é possível mover objetos na árvore alterando a forma inicial do array.

Instalação

// npm
npm i gumga-tree-ng --save

// bower
bower install gumga-tree-ng --save

// Injeção do modulo
angular.module('yourApp', ['gumga.tree']);

// Importação das dependências do componente pelo npm
//-------------------------------------------------------------//
<script src="node_modules/gumga-tree-ng/dist/gumga-tree-ng.min.js"></script>

    angular.module('app', ['gumga.tree']).controller('ctrl', function ($scope, $timeout) {

        $scope.options = {
            events: {
                toggle: () => {
                    console.log("Toggle")
                }
            },
            actions: {
            }
        };

        $timeout(function() {
            $scope.options.actions.toggleAll()
        })

        $scope.pessoas = [
            {
                name: 'Maria',
                filhos: [
                    {
                        name: "Regina",
                        filhos: [
                            {
                                name: "Mateus"
                            }
                        ]
                    }
                ]
            },
            {
                name: "Sonia",
                filhos: [
                    {
                        name: "Lucas"
                    },
                    {
                        name: "Mariana"
                    }
                ]
            }
        ]


        $scope.coloca = () => {
            $scope.pessoas.push({
                name: "Joaquem"
            })
        }


        $scope.adiciona = () => {
            $scope.pessoas[0].filhos.push({
                name: "Teste1"
            })
        }

        $scope.cliquei = ($child) => {
            $child.filhos.push({
                name: "Inserindo"
            })
            console.log($child)
        }

    })
Atributos
Parâmetro Tipo Requerido Descrição
ng-model Object Sim Atributo obrigatório que irá conter o objeto do gumga-tree-ng.
children String String Atributo obrigatório que indica qual atributo do objeto é referente aos sub-objetos da árvore.
options Object Não Atributo que contém ações e eventos do componente. ex: 
$scope.options = {
    events: {
        beforeDrop: (obj) => {},
        afterDrop: (obj) => {},
        beforeDrag: (obj) => {},
        afterDrag: (obj) => {},
        beforeDragLeave: (obj) => {},
        afterDragLeave: (obj) => {},
        beforeDragEnd: (obj) => {},
        afterDragEnd: (obj) => {},
        beforeDragOver: (obj) => {},
        afterDragOver: (obj) => {},
        beforeToggle: (ctrl) => {},
        toggle: (ctrl) => {}
    },
    actions: {
        disableDrag: false,
        disableDrop: false,
        depth: () => {},
        toggleAll: () => {},
        collapse: (obj) => {},
        expand: (obj) => {},
        removeAll: () => {},
        scope: () => {}
    }
  };
Opções e Eventos
Parâmetro Tipo Requerido Descrição
beforeDrop Function Não Evento que é disparado antes de soltar o elemento arrastado.
afterDrop Function Não Evento que é disparado depois de soltar o elemento arrastado.
beforeDrag Function Não Evento que é disparado antes de começar a arrastar o elemento.
afterDrag Function Não Evento que é disparado depois de começar a arrastar o elemento.
beforeDragLeave Function Não Evento que é disparado antes de sair de uma zona de drop.
afterDragLeave Function Não Evento que é disparado ao sair de uma zona de drop.
beforeDragEnd Function Não Evento que é disparado antes de terminar de arrastar o elemento.
afterDragEnd Function Não Evento que é disparado depois de terminar de arrastar o elemento.
beforeDragOver Function Não Evento que é disparado antes de passar sobre outro elemento.
afterDragOver Function Não Evento que é disparado depois de passar sobre outro elemento.
toggle Function Não Evento que é disparado quando abrir ou fechar uma ramificação da árvore.
disableDrag Boolean Não Atributo que permite desabilitar o drag do gumga-tree-ng.
disableDrop Boolean Não Atributo que permite desabilitar o drop do gumga-tree-ng.
depth Function(<elemento>) Não Função que permite verificar o nível de algum elemento.
toggleAll Function Não Função que permite alternar se todas ramificações estão abertas ou fechadas.
collapse Function(<elemento>) Não Função que permite fechar a ramificação do elemento.
expand Function(<elemento>) Não Função que permite abrir a ramificação do elemento.
removeAll Function(<elemento>) Não Função que remove todos elementos da árvore.
removeAll Function Não Função que retorna o escopo interno da árvore.
